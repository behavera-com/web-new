# Case study photos

Drop the 5 background photos here, named exactly:

- `prusa.webp`
- `expando.webp`
- `valxon.webp`
- `vodafone.webp`
- `365bank.webp`

Recommended: 1200×1500 (portrait) for the three top cards, 1600×1100 (landscape) for the two featured bottom cards. WebP, ~80 quality, sRGB.

If a file is missing, the card falls back to a per-brand gradient (see `FALLBACK_GRADIENTS` in `src/components/startupjobs/sections/CaseStudies.tsx`) so the layout stays intact.
